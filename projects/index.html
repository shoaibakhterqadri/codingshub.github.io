Encryption Process is used to keep original content in different format so that no one read and understand. Decryption Process is used to bring original content and display user's original content. Online Hash Function

Onling Hashing Function are SHA-1, SHA-256, SHA-384 and SHA-512 etc. These all are used to hash message or file into different words and these words are not readable. One of the most important quality of Hashing Function is that this is not decrypt. By this way, no one can know about its original content.
